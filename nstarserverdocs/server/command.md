# 服务器指令

## 传送相关

::: tip 冷却时间

无论使用什么指令传送，都会有5分钟的冷却时间！

:::

### 地标传送

传送到出生点：

```
/warp spawn
```

::: tip TIPS

出生点可以使用64绿宝石来自己开一个商店！

:::

### 玩家互传

#### 发出传送请求

请求传送到其他玩家：

```
/tpa 玩家ID
```

请求把别人传送到自己那里：

```
/tphere 玩家ID
```

#### 处理传送请求

接受传送：

```
/tpaccept
```

拒绝传送：

```
/tp
```

## 点歌相关

搜索歌曲：

```
/music search 歌名
```

投票切歌：

```
/music vote
```

自己不再听歌：

```
/music nomusic
```

自己不想听这首歌：

```
/music stop
```
