# 在验证系统注册

1.首先打开浏览器.输入

https://auth.nstarmc.cn (opens new window)

或者可以输入https://auth.nstarmc.cn/auth/login (opens new window)进入注册界面

2.输入自己的邮箱，密码，游戏名称(全英文，符号只允许下划线)

如何获取邀请码？

在QQ群找管理员/群主并提供以下信息

以下为模板，复制后填写发给管理员/群主即可

申请服务器邀请码
邮箱：
将要注册的游戏ID：
1
2
3

（来自小宇的留言：没事别找我）

3.注册完后，你会收到激活邮件，打开你注册用的邮箱，点击链接激活即可！

注意！

如果不从邮件激活账户，你将无法登录游戏！
